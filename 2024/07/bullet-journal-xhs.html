子弹笔记：试着这样记笔记 原来生活这么有趣

发表于 2024-07-20   |   更新于 2024-07-21 | 分类于 Reading | | 阅读次数:
字数统计: 3,823

记笔记是一件很有仪式感的事情，一旦开始了就难以停下来的节奏。请相信子弹笔记术的复利效应，改变往往发生在潜移默化之间，体验进步比想象结果更加有趣。

道：子弹笔记价值观

先谈价值观，再学方法论。

  • 价值观：以有限精力去学习、管理及运用你所掌握的知识，并以目标驱动、阶段闭环、优先次序的方式去记录与做事，最终达成目标、养成习惯、取得佳绩。除此之外，通过定期回顾、复盘知识以及总结经验，以反过来指导及增援未来的自己。双向闭环系统，进步永不停歇。

  • 方法论：结合注意力、生产力、意向性，搭建一个集时间、精力及知识管理的笔记系统。

    • ⚠️ 注意力：人的注意力非常有限，高效管理精力、合理投资时间，要求我们必须高度聚焦做事情。
    • 📚 生产力：知识变现转化为生产力，在认知以内创造无限价值。
    • 🚩 意向性：设立具有进步意义的目标，驱动着我们干事创业，而非表演性质随机游走做事情。

万事开头难，迈出第一步才有机会发生故事，子弹笔记术教你从整理想法开始。

  • 想法容易脱离掌控，易于消逝混沌之中。及时捕捉并记录想法，才能储备足够多的素材，以供在冷静时候审视它们。
  • 想法总是转瞬即逝，把消耗注意力的念头、选项逐一写下，得到一张思想清单。

思想清单，记录每天正在做的、近期应该做的以及未来想要做的事情。为了系统性地掌控局面、丰富想法的可塑造性，建议融入更多维度来描述它。

  • 按意义划分：创业、考试、健身
  • 按时间划分：短期、中期、长期
  • 按频次划分：期限性、周期性

总的方法不变，结合 正在做的、应该做的以及想要做的 x 不同维度主题划分，发挥乘法效应，因地制宜设计个性化的思想清单（也称集子）。个性化集子，要具备主题性、故事性、目的性，比如记录长期主义事业、每日感恩集锦等。如此一来，在子弹笔记术指导之下，创造性地产出各式各样的笔记集子。比如：

  • 🗂️ 规划本：长期主义事业、考证之路、人生三大事
  • 📒 清单集：读书清单、美食探店、败家愿望
  • 💗 感恩集：亲密关系、职场教训、思想顿悟
  • 📚 知识库：专业知识、街头智慧、投资理财
  • 📮 收件箱：未分类、临时的任务、事件或笔记
  • 🔮 自由发挥：灵感启发、物品登记、观影清单……

有了较完整的思想体系以及笔记框架，接下来便是具体的实践方法与指导思想讲解。过程中一定要动手实践，不断尝试迭代、复盘总结以及改进创新，以形成最适合于自身的子弹笔记术。

器：子弹笔记结构体

概念解释

  • 模板：子弹笔记的基本组成单元，每个集子都可套用该模板进行创作。

    • 目录索引：标题 + 页码，用于定位内容、定位集子。
    • 未来记录：记录本月以外的任务和事件，用于记录带有长期主义色彩的事情。
    • 月度记录：记录本月的任务以及时间安排（或本月思想清单），或从总目标中拆解阶段性（中长期）目标。
    • 每日记录：记录每天待办事项，以及不时产生的零碎想法。

  • 集子：集子继承了主体笔记的组成单元，即包括索引、未来记录、每月记录以及每日记录。

    值得注意的是：
    1) 集子与主体笔记的风格形式保持一致，也可以进行个性化设计。个性化集子应具备主题性、故事性、目的性，比如记录每日感恩集锦、心愿物品清单等。
    2) 模板只是一种参考，类似单纯用于记录静态信息的集子就没必要套上未来、月度及每日记录了。

  • 速记：

    • 利用统一的简短记号或符号（Markdown、Emoji、颜色体系）快速捕捉想法。
    • 为了操作可行性，形式大致可分为任务、事件、笔记三类速记模板。

  • 迁移：

    • 每天或定期从收件箱中回顾与归档信息。
    • 每月从笔记中过滤掉无意义内容的过程。
    • 零碎知识体系化之后，即时清理掉它们。

笔记系统

  • 子弹笔记术，就像包罗万象的生态系统，各种提升效率的技巧与哲思集于一体。

  • 个性化集子，既延伸了子弹笔记的功能用途，也在笔记系统中扮演着重要有机一员。比如：

    • 📮 收件箱：缓存速记下来的任务、时间、笔记，再通过分发迁移至目标集子。
    • 📚 知识库：外挂的“第二大脑”，记笔记不在于记录本身，而是知识内化的过程。知识内化本质是增援未来的自己。

    • 🗂️ 规划本：撰写事业或项目的背景、目标计划、时间规划、主观感悟与复盘总结，大篇幅文字描述的内容记录在此集子。

      用于规划管理的规划本

    • 🗓️ 任务单：从规划本中抽取待办项，事项表述主题及关键字，无须大篇幅赘述。过程中一边执行任务一边量化指标，形成用于项目管理的集子。

      用于行动管理的任务单

    • 📒 清单集：记录静态信息的集子。

个性集子

  • 计划集子：
    • 凡事预则立，不预则废。
    • 研究不宜过于深入，投入时间越长，得到选择越多。毕竟再充足的事前调研也避免不了突发情况，随机应变、及时行动，成事的机率更高。
  • 清单集子：比如，购物清单、网红打卡、阅读清单等。
  • 行程集子：比如，旅游行程、环岛骑行、工程实践等。
  • 追踪集子：比如，读过的书、看过的电影、美食探店等。
    • 只关注结果，会忽视过程中浮现的有价值信息，追踪记录的关键不仅在于观测进步，而是在于培养自我意识。
    • 事物间的关联性并不是显而易见的，但只要持续的记录就一定能找到内在联系，因果才至关重要。
    • 了解的越多，行动就越有效。

法：子弹笔记方法论

记笔记意义在于不断改变现状、成就更好自己。“道” 篇章让我们确定到了方向，“器” 篇章让我们具备了记笔记的框架及工具，“法” 篇章在于灌输用于指导行动的方法。

道法器三者有机统一，共同构成一个较完整的思想体系和实践框架，过程中不断迭代、复盘总结以及创新改进，终究形成最适合于自身的子弹笔记术。

总则

  • 摆脱忙碌：忙碌是一种疯狂循环状态，疯狂循环消耗太多注意力，从而限制了我们感受、成长与思考的能力。打破这种死循环以重复新生 。
  • 课题分离：把遇到的事和做出的回应分隔开，审视自己的经历，了解哪些能掌控的，什么具有意义，什么值得留意。
  • 信仰行动：信仰与行动有机联系在一起，带着意向性去生活，有所收获的同时找到意义。

反思

  • 自省吾身，常思己过，善修其身。
  • 反思目的不是对过去犯的错误进行惩罚，而是要从过去的经历中涉取信息养分滋养未来成长。
  • 每日反思
    • 被动回应：受到刺激 —> 被动回应 —> 通常会感到恐惧焦虑
    • 意向驱使：受到刺激 —> 主动应对 —> 通过选择/方法/目标 —> 人会积极反思
  • 迁移月度反思和年度反思：
    • 迁移操作实质上是一种过滤机制，你需要对未完成的任务进行反思，如此迁移才有意向性。
    • 重新思考未完成任务，究竟是以什么样的角色、地位、优先次序参与下月或者明年的计划。

意义

  • 思想实验：试想自己站在岔路口，面前有两条路可走，一条人迹更多，另一条人迹更少。想象迟暮之年，在个人生活或者专业技术方面会获得什么成就，这样的生活给您带来什么结果？
    • 人迹更多的路：带你走向更熟悉的地方，是目前生活的延续，只追求舒适的进步，不降低现有生活品质，也不改变现状。
    • 人迹更少的路：带你走向不熟悉的地方，冒险多于舒适的生活。敢于追求感兴趣的目标，愿意通过奋斗来提升自己。
  • 身后反思：分别为两条路创建一份 “两种生活” 的集子。选择你喜欢的生活方式，圈出你最骄傲的成就。完成后把这些事项迁移到你的目标集子。

目标

  • 54321倒计时法则
    • 将目标分为短期、中长期、长期。
    • 空白左右两页，左页个人目标，右页职业目标，个人事业与职场事业分开谈，两边分别投入精力发展。
    • 每个页面分 5 栏，5 栏依上往下是 5 年内目标、4 月内目标、3 周内目标、2 日内目标以及 1 小时内目标。
  • 分清目标主次轻重
    • 每一栏只考虑一项最先考虑的目标。
    • 注意力残留原则，一次只专注于一件事，专注于实现目标过程。最具价值的也是实现目标的过程，因它占据了大部分的生活，因此提供了大量信息让你成长。
  • 长期目标为冲刺目标
    • 并不一定要把喜欢的事情变成职业，建议在体验过一段时间后再做决断。
    • 将长期目标分解一个短期、独立的冲刺目标。类似软件开发的开发周期，分阶段设立目标。
  • 设立冲刺目标的要求
    • 万事开头难，初始阶段忌设立大障碍目标
    • 阶段性取胜，具有清晰、可行的任务步骤
    • 拖延是敌人，必须固定在短期时间内完成

时间

  • 时间箱管理：把任务完成时间规定在一定范围内，其意图是把百分百的注意力按照规定的时间限制投入一事物上。

    两个关键激励因素：条理感和紧迫感

  • 勿拖延请置顶：找出自己最高效、最专注的时候，事情从难到易排序，以制定执行计划。

循序渐进

  • PDCA 循环：Plan -> Do -> Check -> Act
    • 计划 / Plan：发现机会，制定改进计划
    • 执行 / Do：执行计划，测试改进结果
    • 检查 / Check：分析测试结果，确认学习成果
    • 行动 / Act：按照学习成果行动
  • 原则上一次计划指导一次实际行动。
    • 无改进：换一个计划重新开始，循环上述步骤
    • 有进步：结合所学执行新计划，循环上述步骤
  • 大事化小事：任务尚存难度，说明任务拆得不够细。
  • 达成小成就：找到任何一种成就，即使微小都无所谓，降低标准保证切实可行，然后以任务子弹的形式写进子弹笔记，完成它以达成成就。

感恩

  • 自我庆祝：根据自己心理预期，对所有事项设定奖励机制，有意识地创造动力以形成良性循环。
  • 感恩子集：创建感恩记录，并每天写下至少一件让你感恩的事。任何经历都是有意义的，对生活的留心观察，微小的幸福汇将促成更美好的生活。
  • 越欣赏，越懂欣赏。随着不断积极寻觅生活中的美好，提升发现美的能力，便能更好地发现美、欣赏美。

掌控

  • 我们无法掌控外部的世界、人和事，但我们能控制自己的内心世界。

  • 应对与回应：

    • 若与他人有冲突之时，冷静分析现状，千万不要有过激行为，毕竟恐惧或愤怒的回应只会加深对方和自身的创伤，失去相互理解、解决问题的机会。
    • 因此，需要每日反思重新审视这件事。

  • 过程与结果：

    • 日反思与夜反思以及月度迁移时，遍历任务列表，试着找出在你掌握之内以及之外的事情。
    • 无法掌控的事情要舍得放手。

  • 观察这条任务是侧重结果，还是侧重于过程。尽管目标带来方向，若目标过于侧重结果，往往结果是不可控的。

    例：减脂 10%、雅思 6.5、做一个很棒的汇报等。

  • 对于过于庞大的目标，尝试把目标分解成一个个可执行的小步骤。

拆解

  • 五问法：分解大问题为小问题，在子弹笔记中创建一个子集，以问题为标题，以不同维度提出问题，并挖掘潜在的价值。

辐射

  • 自我怜悯：自我评判与自我反思。
  • 坚强信念：坚信你的一言一行所带有的能量会改变周遭世界和其中的人。
  • 共同进步：辐射力是双向的，自己与朋友自己都需要时不时接受对方的审视。找一个朋友，两人互相尊重、赏识、关怀，共同成长。

忍耐

  • 价值升华：不要把注意力放在干活的痛苦一面，我们更应该关注它带来收获的方面。创建一个分析记录，留意为什么要做这件事情，努力的具体目的是？

    例如：付房租 —> 浪费钱？ <—> 学区房 / 通勤时间短 / 朋友相聚 —> 平衡心中的负面情绪

  • 追踪进度：不要轻易放弃，但一件事情对于自己目前来说毫无价值，或是付诸努力与结果不成正比，这可能是分散注意力的事，暂且放下它吧。 利用月度迁移定期检查自己的进度。

无力感

  • 橡皮鸭调试法：如身边无人聆听，你可以向一只玩具橡皮鸭写一份信，在详细叙述事情的过程中，可能你已经找到应对方法了。

    • 你遇到的问题
    • 出问题的地方
    • 出问题的原因
    • 尝试过的方法
    • 未尝试的方法
    • 你想要的结果

  • 叙事过程即思路梳理过程，灵感启发往往在于掌握事情来龙去脉的前夕瞬间。
