佛畏系统：借以系统思维助我走出人生无序状态

发表于 2024-01-14   |   更新于 2024-01-28 | | 阅读次数:
字数统计: 5,776

凡夫畏果，菩萨畏因，佛畏系统。

  • 先谈两种认识世界的思维模型：

    • 因果思维：借助因果去认识世界，理解事物因果，让人未来有打算、做事情有预案，不至于对什么都感到迷茫。理解因果关系，不沉溺于事后结果，执因索果，透过现象看穿事物本质。然而，真实世界是复杂的，事物发展结果可能由多个归因导致，也有可能因果关系环环相扣，说不清道不明起因、主因亦是常事，何况真实世界还充满了不确定性，即因果知识不能保证我们全然掌控局面，从而科学地做决策。
    • 系统思维：系统性思维，让我们可以宏观地、辩证地、长远地看待每一事物的发展趋势，抓系统规律、辨共性问题、设计系统解决方案，最终做出高质量决策以引导事物往积极方向发展。系统是一个整体、一个组织、一个框架，允许事情在一定范围内变化，怎么变脱离不了一定框架。拥有系统，每发生的一件事只不过是一个个例、一个数字而已。面对一系列相似的问题，都可建立系统去应对解决。

  • 系统思维有助于提升我们认识、理解和解决问题的能力，具体实操又分两方面去执行：

    • 借鉴他人系统：本书讨论了五大类系统，职业发展系统、学习研究系统、日常做事系统、情感人生系统以及社会大系统。借鉴本书建议直接上手实操与验证，比如职场生涯如何规划发展、怎样做好一个领域的研究、怎样摆脱日常事务的枷锁等。
    • 创造个人系统：每个人都需要系统思维，书中方法只是一个引子，将来遇到的每一类问题都可建立一个应对系统。初创系统只是 1.0，未来还要不断打磨这套系统，以致于在面对不确定性时，有方法、有底气做出属于自己的科学决策。

职业发展系统

四种好运

  • 祝君拥有四种好运：人的命运要靠自我奋斗，但也要考虑历史的行程。行程中，运气与自我奋斗同样重要，运气是改变命运的重要一部分。运气看似遥不可及的事物，事实上它就存在于身边，要予以区别对待，抓住可控部分即有获得运气的可能性。

    祝君拥有四种好运

    • 盲目的随机性：福利彩中大奖、微博转发锦鲤、雍和宫求姻缘等，这就是纯随机性的运气。

      启示：该类型运气可遇不可求，未得到前不气馁，做好认知准备；有幸得到要懂得感恩。

    • 跑出来的机会：经历足够久、尝试足够多，才得到一个可行的事实。

      启示：大数定律，在取得更多数据样本前，得多体验、善总结，不要过早给事情打标签、下定论。

    • 有准备的头脑：机会只青睐有准备的头脑，就像近一年兴起的 AIGC，机会摆在所有人面前，但只有愿意 “下马看花” 的人才能看到它的价值。这是需要知识的好运，而且它还会带来复利效应。

    • 人设的吸引力：吸引力法则，自动找过来的好运，而它来源于你身上某种特殊性。稀有事物往往要求极其特殊的条件。

      启示：自我奋斗与历史行程的共同产物。每个人都有他的出场时机，足够努力、无限进步，“命运” 自有安排。

  • 重大成就往往是四种好运的综合性结果，一点随机性，一点主动性，一点客观性以及一点特殊性。想明白哪些是取决他人的，哪些是可争取的，哪些是须等待的，哪些是自己的，也许我们对命运多掌握一点主动权。

秘密项目

  • 上班之余，私下干一件大项目。这个项目非业务爱好，得认真严肃对待，每天有进展，保持进步性，最终达到很高水平。
  • 或许你会有疑问，好项目为啥要秘密地做，全职做不是更好吗？关键不是全职还是兼职，而是要有一点疏离感。疏离有别于一群人干活干事、头脑风暴，而是独立思考以激发创造性。

  • 创造更像是想法的连接：

    • 创造性活动本质上是一个混圈子的事情，越遥远、越微弱的连接，往往越有意思。

      圈子泛指跨文化圈、跨能力圈。

    • 关注来自边缘的连接，虽不太可能融入圈子，但他们能帮着扩大圈子。

X 因素

  • X 因素：泛指某种东西身上解释不了某种因素。另一个含义，它能让人们为这个因素容忍某事物所有缺点。
  • 以产品角度讨论，包含 X 因素的产品像是藏有惊喜设计的最小可行产品（Minimum Viable Product, MVP）。
    • 惊喜设计，会让用户眼前一亮，甚至可容忍其他缺陷存在。
    • 最小可行，以最小成本验证产品是否有用、有市场，有则优化迭代，无则淘汰重来。
  • 以成功者角度讨论，那些看似有很多缺点但又特别成功的人，其中必定有个很强的 X 因素。X 因素是能让你超越平庸的东西。
    • 启示 1：我们总是在追求学习有用的知识，并追求学得极致好。但学习好并不代表你能创造出众的事物，大概率情况下只能达到标准的、合格的、平庸的水平。何况他人也可复制路径模子以做到同样的事情。
    • 启示 2：如何打造个人 X 因素，首先要明白一个道理，社会分工把人物化、人力资本化，大多数人都是应试教育培养出来的通识性专业人才，人力资源具有很强的可替代性。与其埋怨无意义的内卷时代，改变不了的劳动力市场，我们能做的就是改变自身供给能力，从而在严重同质化的劳动力市场中获得竞争力。
    • 启示 3：从容应对市场供需变化，眼光还得放得远一些。个人如何异军突起、冲出重围，将是我们必须且中长期都要思考的课题，比如跨学科思维修炼、稀缺专业能力培养、避免消费主义等。

扮演法

  • 扮演法之窍门：

    • 扮演哪种职业，就得遵守哪种人的行为规则，不套公式、随机应变，要演的像。

      比如，当官要有官员样，永远稳当，胸有成竹，不轻易表达，不暴露弱点，关键时刻有所担当，时刻盯住高层动向。

    • 不要混淆扮演的角色和自我的存在，演着演着就忘记本我了。

      扮演法会影响“非凡者”自我意识的养成，让他们变得越来越不像自己，甚至被残留假象同化。系统是需要个性存在的，好的个性在系统中保留下来并引导系统向积极方向演化。

    • 抵抗失控的诱惑，不忘记初心。

  • 不同段位的非凡者，面临的困境也不同。

    • 初级非凡者，出戏太多，演得不像，晋升很慢。
    • 中级非凡者，演技出众，入戏太深，危险在于为了晋升而不择手段。
    • 高级非凡者，忘记本我，没有了个人意识，一懈怠就得面临失控的风险。

权利机制

  • 有能力不一定能改变世界，但拥有权利可以，至少可改变你权利范围之内的世界。
  • 取得权利的四条路径
    • 强势支配行为：以高压姿态直接恐吓、打击或压迫，逼着别人听你的。这是动物界取得权利直接有效的手段。在权利斗争中，过于文明会吃大亏。
    • 政治行为：一个人时唯我独大，二个人时会有斗争，三个人时就有政治。分清楚谁是朋友，谁是敌人，团结可团结的力量，统一战线以取得胜利。
    • 公益行为：先实现他人价值，才体现自己价值。在别人有困难时主动伸出援手，或许人设的吸引力就是这样一点一点创造出来的。
    • 声望行为：也称为能力行为，能给集体、团队带来好处的能力。但光有能力也不行，不作为不提升团体的声望，这是德不配位。

寻宝思维

  • 职员思维：优等生心态害怕犯错误，认为所有的失败都应受指责，所有的浪费都应该反省。优等生心态实际是职员思维，做事情要明明白白、必须安排好的。
  • 寻宝者思维：当今不知道做哪件事，做每件事无法预知后果等，好比一个企业家、投资人或领导者，你得从一堆不确定性中获利或达成某成就，此刻你需要换一个系统，以寻宝者思维替换职员思维。
  • 人的一生之中，影响最深的可能是几本书，决定前途的可能是一两项技能，改变命运的可能是几件事情，人生贵人也许就那么几个。错过他们的损失不可估量，识别机遇、抓住机会，我们要把主动权攥在手里。
    • 其一，选择了潜在好东西，并为之努力奋斗，并争取做到最好。这需要提高判断力和做事能力，为了提高成功率，需要学习和模仿，行业内顶级大佬就是最好的老师。
    • 其二，必须放量，别指望奇迹。量变是质变的原始积累，并且每一份量变都具有意义。

学习研究系统

调研的心法

  • 提问：好的调研必须从提问开始。区别 “问题” 与 “提问”，把问题变成一个可调研的提问。

    问题是隐性的，难以言传；提问必须是显性的，强指向性。

  • 逻辑脉络：脑子有张脉络清晰的调研路线图。真实情况，对于某问题的提问都是嵌套的，针对不同提问展开不同调研，每一步调研填补一个逻辑空白，最后 “串点成线”，以找到解决问题的方法。

  • 串点成线：搞清楚问题，暂停调研，将调研结果用自己的语言组织成一份报告。以逻辑顺序、演变顺序、时间顺序等，任君选择，只要报告是自洽的、完备的即可。

高级调研法

  • 调研要做到这个领域最先进水平，这里的调研不是为了全面了解一个领域，为了这领域编写百科全书。高级调研法在于找到独特的定位，发现它最前卫、最活跃、最有意思的地方。

  • 过程分五个步骤执行：

    • 读书：选定某领域展开学习时，建议找其中 5 本书来阅读。
      • 畅销书：通俗易懂，迅速了解基础知识
      • 热门书：技术性强，之所以成为热门，总有让读者感到兴奋和刺激的方面。把其中最喜欢的、最能激发想象力的方面记录下来，它们有助于你去寻找独特定位。
      • 专家视角的书：技术性强，学习专家是如何考虑问题的，带来大局观念，开拓我们的视野。
      • 有深度的硬书：写给专家看的书，点拨该领域真正的难点所在。
      • 未来走向的书：关于某领域未来走向的书，感受最前沿的东西。
    • 采访专家：读书总会遇到问题，解决问题最便捷的途径便是请教领域的专家。
    • 到达前沿：前沿没有公认确定的答案，属于教科书上的空白，这也恰是我们能有所作为的地方。
    • 行成观点：反复考虑各方意见，自己拿定主意。

    • 完成叙事：调研过程中，把学习到的知识总结成篇，以新的观点讲成一个故事。这意味着要把该事情的因果关系理顺，并建立起一个思维模型。

      启示：如何讲好故事，这得借助教学相长的费曼学习法去宣讲讲授，别人听明白了证明你学到位了。　

初学者姿态

  • 智识谦逊是一个思想状态、一种思想境界。智识谦逊的人愿意接触新事物，愿意学习新知识，即大脑在心理学上是可塑的。
  • 智识谦逊者共有特征：
    • 喜欢学习新知识，对科学很感兴趣。
    • 善于从多个视角考虑问题，能采纳他人视角看待问题。
    • 愿意跟人交往，会倾听他人的声音。
    • 咨诹善道，察纳雅言。
  • 头脑封闭者共有特征：
    • 对新知识不感兴趣，只喜欢符合自己认知的信息。
    • 思维定式，手持锤子看什么都是钉子。
    • 先看立场后看观点，甚至不顾事实，对政治议题的看法越来越极端。
    • 陷入恶性循环，认识世界非黑即白。
  • 技艺的真正秘诀在于永远当一个初学者。对自己的知识水平有正确认识，多给别人讲讲，讲授过程中说不明白的，说明你并不真正理解它。

  • 主动提取不同的声音，避免陷入 “确认偏误”。就像重大决策需要设立一个反方，反方提供不同观点、相反意见。

    逆向性拷问：做决策，相较于必要性论证，多找不去做的理由，反证不得不去做的原因。

日常做事系统

不要被小事击垮

使人疲惫的不是远方的高山，而是你鞋子里的一粒沙。 —— 伏尔泰

  • 生活中击垮我们的往往不是什么惊天动地大事，而是叠加在一起琐碎、微不足道的事情。
    • 慢性日常麻烦：让人有压力感，压力感会让人体一种皮质醇的荷尔蒙浓度升高，进而破坏免疫系统。免疫系统不好使，身体容易出现慢性炎症，进而导致更严重的疾病，长期炎症会增加得癌症的风险。
    • 对麻烦的感知和反应：小麻烦永远存在，麻烦本身不影响身心健康，而是我们面对麻烦时的反应。
    • 怎样应对麻烦：弱者报复，强者原谅，智者忽略。
      • 善于观察自己的情绪。要爆发情绪时，利用 “腹式呼吸法 + 冥想” 方式让自己冷静下来。
      • 多关注、多记录开心的事，挤占不开心的位置。主动营造小确幸的时刻。
  • 事物总有它的两面性，我们要辩证看待它们。一味学忽略小麻烦也是不对的。永远处于战斗状态固然不好，偶尔也要适当承压或暴露在危险当中，反脆弱的成长法则。

放弃必须做的事

  • 时间管理最高境界
    • 四象限时间管理法把事情按照重要和紧急两个维度分类，随着时间推移，人们日程表上紧急的事情会越来越多。
    • 必须要做的事做不完，谈何时间管理。为此，作者认为时间管理最高境界则是放弃必须做的事。日程表都排满，做事只是随波逐流罢了。

  • 没有必须要做的事，你就有了选项。从可选项中选一个心理接受程度最高的。

  • 余闲的价值：余闲代表任何一种多出来、可任意使用的资源，它泛指金钱、时间或空间。

    • 贫穷和忙碌会把人陷入一个特定的思维模式，思想决定行动，它会进一步限制人的发展能力，使得穷人更穷。
    • 拥有余闲的两个好处：
      • 不用必须选择：资源充裕，不用权衡做出取舍。
      • 不害怕犯错误：探索需要代价，创造来自余闲。

  • 别透支时间：如果可把控的时间很少，就别预知未来时间。事情会挤占下一件事的时间，下一件事挤占另一件事的时间，跌入恶性循环。

  • 自由和自律：若已成为时间上的穷人，怎样才能给自己积攒一些余闲呢？
    • 重新启动：干脆放弃一些项目，一次性得到一大笔余闲。比如，主动休假或换个工作。
    • 珍惜余闲：自律给予自由，余闲很容易流失，主动权攥在手中，积极发挥余闲的价值。

十全十美代价高

  • 代价：没有漏洞的系统不见得是最好的系统，最好的系统应该在减小损失比不上付出成本时，停止优化。维护一个没有错误的系统，花费成本很高。
  • 放飞：活得太累时，别怕放飞自我。

  • 试探：”好人” 总是在适应世界的规则，”坏人” 总是在试探世界的底线。

    • 在公司里，上级领导是一个很严厉的人，大家在他面前不敢多说一句话，只好老老实实做事情。
    • 有时候给你的命令是不对的，通过自己的方式实现更好，请积极试探，这样才能把握老板的容忍度。

    • 试探要把握一个度，每次稍微突破一点，观察对方有什么反应，对方能接受就 “得寸进尺”，抗议时马上停下。

      但不是所有场合都适合试探，有关是非问题，底线思维还是得有。

马尔可夫宿命论

  • 定理：马尔可夫过程过程要求满足四个条件。

    • 1：系统中有有限多个状态。
    • 2：状态之间切换的概率是固定的。
    • 3：系统要有遍历性，从任何一个状态出发，总能找到一条路线，切换到任何一个其他状态。
    • 4：没有循环的情况，某几个状态形成闭环，把其他状态排斥在外。

  • 只要是马尔可夫过程，不管初始值如何，也不管过程中有什么一次性的干预，它终究会演化至一个统计的平衡态，其中每个状态所占的比例不变。

  • 四个条件中，只有第二个条件是关键，即状态之间切换的概率是固定的。比如，不发达地区很多人因疾病而不得不去借债，还不上钱就变成贫困户。若是给他们一大笔钱，能否改变现状呢？

    并没有改变他们下一次得病或欠债的概率，你改变的现状仅是一个初始条件，只要概率不变，他们的宿命终究不变。

  • 用途：

    • 马尔可夫模型，适用于那些不随以前历史发生而改变概率的情况。比如，一家公司因一个开明领导，短短几年业绩攀升，但高峰之后又走回老路。这意味着改变公司历史，要改变作用机制，只改变表现形式，那只是状态变化而已。
    • 生活中有些事情是路径依赖的，意味着后面发生的概率会根据之前发生的事情做出改变。 比如，长期抽烟酗酒的人，让他一周不喝酒并不足以改变它抽烟酗酒的概率，但如果想办法让他连续一年不抽不喝，那真可能戒烟戒酒成功。

决策是一门技能

  • 一般人之所以不善决策，根本原因在于 缺少决策的机会。生活中大多数情况只是在走流程，按部就班，不用过多思考，也不用冒风险，为此也 没承受过错误决策的后果。
  • 科学与理性的决策是有方法论可循的：
    • 第一步：看看手中的可选项。
    • 第二步：评估各选项的价值，考虑机会成本，选择价值最高的选项。
    • 第三步：执行过程中，根据实际情况动态调整。

  • 列出自己的所有选项：

    • 切勿认准一条路一直走到黑，启示还有很多备选项。

      • 建议你把所有选项写下来，比如画思维导图。思维导图真正作用是把各种选项直观地呈现在你面前，帮助决策。
      • 参与决策的不止一个人，头脑风暴是个好方法。大家聚在一起集思广益，先别管可不可行，想法提出来，给大家一个启发。

      • 决策要参考多样性的意见，多样性指 视角和思维模型 的多样性，而不是 利益诉求 的多样性。

        决策参与者为了共同目标走到一起，而不能是一群互相之间充满利益冲突的人，否则决策就成为一群人对另一群人的打压。

  • 选择价值最高的选项：

    • 高明决策者会让一些人从反方面寻找证据，力图推翻自己的决定。如此一来，你要找到充分理由去说服反对者，经过此过程推敲，决策才算是成熟的。
    • 未来具有不确定性，决策总包含着预判，甚至含有赌博成分。但世界还是讲理的，多看看别人做类似事时结果如何，多一份输入多一点成功机会。
  • 执行过程中随机应变：
    • 把决策参与者与决策拍板者分离开来。高水平决策机制，不应该是刘备与诸葛亮的关系，而是曹操与其智囊团的关系。把多样性意见摆在一起综合判断。
    • 把决策过程与决策结果分离开来。决策具有不确定性，决策与结果没有必然导致关系，为此我们不能根据结果好坏去评价决策水平。科学的决策系统追求的不是每一次都赌赢，而是让赢的概率大于输的概率。
    • 把决策贡献与决策身份分离开来。探讨问题不分年龄、文化水平等，团队文化影响着决策水平。
