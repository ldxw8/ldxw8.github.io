佛畏系统：借以系统思维助我走出人生无序状态

发表于 2024-01-14   |   更新于 2024-01-27 | | 阅读次数:
字数统计: 2,624

凡夫畏果，菩萨畏因，佛畏系统。

  • 先谈两种认识世界的思维模型：

    • 因果思维：借助因果去认识世界，理解事物因果，让人未来有打算、做事情有预案，不至于对什么都感到迷茫。理解因果关系，不沉溺于事后结果，执因索果，透过现象看穿事物本质。然而，真实世界是复杂的，事物发展结果可能由多个归因导致，也有可能因果关系环环相扣，说不清道不明起因、主因亦是常事，何况真实世界还充满了不确定性，即因果知识不能保证我们全然掌控局面，以做出科学的决策。
    • 系统思维：系统性思维，让我们可以宏观地、辩证地、长远地看待每一事物的发展趋势，抓系统规律、辨共性问题、设计系统解决方案，最终做出高质量决策以引导事物往积极方向发展。系统是一个整体、一个组织、一个框架，允许事情在一定范围内变化，怎么变脱离不了一定框架。拥有系统，每发生的一件事只不过是一个个例、一个数字而已。面对一系列相似的问题，都可建立系统去应对解决。

  • 系统思维有助于提升我们认识、理解和解决问题的能力，具体实操又分两方面去执行：

    • 借鉴他人系统：本书讨论了五大类系统，职业发展系统、学习研究系统、日常做事系统、情感人生系统以及社会大系统。借鉴本书建议直接上手实操与验证，比如职场生涯如何规划发展、怎样做好一个领域的研究、怎样摆脱日常事务的枷锁等。
    • 创造个人系统：每个人都需要系统思维，书中方法只是一个引子，将来遇到的每一类问题都可建立一个应对系统。初创系统只是 1.0，未来还要不断打磨这套系统，以致于在面对不确定性时，有方法、有底气做出属于自己的科学决策。

职业发展系统

四种好运

  • 祝君拥有四种好运：人的命运要靠自我奋斗，但也要考虑历史的行程。行程中，运气与自我奋斗同样重要，运气是改变命运的重要一部分。运气看似遥不可及的事物，事实上它就存在于身边，要予以区别对待，抓住可控部分即有获得运气的可能性。

    祝君拥有四种好运

    • 盲目的随机性：福利彩中大奖、微博转发锦鲤、雍和宫求姻缘等，这就是纯随机性的运气。

      启示：该类型运气可遇不可求，未得到前不气馁，做好认知准备；有幸得到要懂得感恩。

    • 跑出来的机会：经历足够久、尝试足够多，才得到一个可行的事实。

      启示：大数定律，在取得更多数据样本前，得多体验、善总结，不要过早给事情打标签、下定论。

    • 有准备的头脑：机会只青睐有准备的头脑，就像近一年兴起的 AIGC，机会摆在所有人面前，但只有愿意 “下马看花” 的人才能看到它的价值。这是需要知识的好运，而且它还会带来复利效应。

    • 人设的吸引力：吸引力法则，自动找过来的好运，而它来源于你身上某种特殊性。稀有事物往往要求极其特殊的条件。

      启示：自我奋斗与历史行程的共同产物。每个人都有他的出场时机，足够努力、无限进步，“命运” 自有安排。

  • 重大成就往往是四种好运的综合性结果，一点随机性，一点主动性，一点客观性以及一点特殊性。想明白哪些是取决他人的，哪些是可争取的，哪些是须等待的，哪些是自己的，也许我们对命运多掌握一点主动权。

秘密项目

  • 上班之余，私下干一件大项目。这个项目非业务爱好，得认真严肃对待，每天有进展，保持进步性，最终达到很高水平。
  • 或许你会有疑问，好项目为啥要秘密地做，全职做不是更好吗？关键不是全职还是兼职，而是要有一点疏离感。疏离有别于一群人干活干事、头脑风暴，而是独立思考以激发创造性。

  • 创造更像是想法的连接：

    • 创造性活动本质上是一个混圈子的事情，越遥远、越微弱的连接，往往越有意思。

      圈子泛指跨文化圈、跨能力圈。

    • 关注来自边缘的连接，虽不太可能融入圈子，但他们能帮着扩大圈子。

X 因素

  • X 因素：泛指某种东西身上解释不了某种因素。另一个含义，它能让人们为这个因素容忍某事物所有缺点。
  • 以产品角度讨论，包含 X 因素的产品像是藏有惊喜设计的最小可行产品（Minimum Viable Product, MVP）。
    • 惊喜设计，会让用户眼前一亮，甚至可容忍其他缺陷存在。
    • 最小可行，以最小成本验证产品是否有用、有市场，有则优化迭代，无则淘汰重来。
  • 以成功者角度讨论，那些看似有很多缺点但又特别成功的人，其中必定有个很强的 X 因素。X 因素是能让你超越平庸的东西。
    • 启示 1：我们总是在追求学习有用的知识，并追求学得极致好。但学习好并不代表你能创造出众的事物，大概率情况下只能达到标准的、合格的、平庸的水平。何况他人也可复制路径模子以做到同样的事情。
    • 启示 2：如何打造个人 X 因素，首先要明白一个道理，社会分工把人物化、人力资本化，大多数人都是应试教育培养出来的通识性专业人才，人力资源具有很强的可替代性。与其埋怨无意义的内卷时代，改变不了的劳动力市场，我们能做的就是改变自身供给能力，从而在严重同质化的劳动力市场中获得竞争力。
    • 启示 3：从容应对市场供需变化，眼光还得放得远一些。个人如何异军突起、冲出重围，将是我们必须且中长期都要思考的课题，比如跨学科思维修炼、稀缺专业能力培养、避免消费主义等。

扮演法

  • 扮演法之窍门：

    • 扮演哪种职业，就得遵守哪种人的行为规则，不套公式、随机应变，要演的像。

      比如，当官要有官员样，永远稳当，胸有成竹，不轻易表达，不暴露弱点，关键时刻有所担当，时刻盯住高层动向。

    • 不要混淆扮演的角色和自我的存在，演着演着就忘记本我了。

      扮演法会影响“非凡者”自我意识的养成，让他们变得越来越不像自己，甚至被残留假象同化。系统是需要个性存在的，好的个性在系统中保留下来并引导系统向积极方向演化。

    • 抵抗失控的诱惑，不忘记初心。

  • 不同段位的非凡者，面临的困境也不同。

    • 初级非凡者，出戏太多，演得不像，晋升很慢。
    • 中级非凡者，演技出众，入戏太深，危险在于为了晋升而不择手段。
    • 高级非凡者，忘记本我，没有了个人意识，一懈怠就得面临失控的风险。

权利机制

  • 有能力不一定能改变世界，但拥有权利可以，至少可改变你权利范围之内的世界。
  • 取得权利的四条路径
    • 强势支配行为：以高压姿态直接恐吓、打击或压迫，逼着别人听你的。这是动物界取得权利直接有效的手段。在权利斗争中，过于文明会吃大亏。
    • 政治行为：一个人时唯我独大，二个人时会有斗争，三个人时就有政治。分清楚谁是朋友，谁是敌人，团结可团结的力量，统一战线以取得胜利。
    • 公益行为：先实现他人价值，才体现自己价值。在别人有困难时主动伸出援手，或许人设的吸引力就是这样一点一点创造出来的。
    • 声望行为：也称为能力行为，能给集体、团队带来好处的能力。但光有能力也不行，不作为不提升团体的声望，这是德不配位。

寻宝思维

  • 职员思维：优等生心态害怕犯错误，认为所有的失败都应受指责，所有的浪费都应该反省。优等生心态实际是职员思维，做事情要明明白白、必须安排好的。
  • 寻宝者思维：当今不知道做哪件事，做每件事无法预知后果等，好比一个企业家、投资人或领导者，你得从一堆不确定性中获利或达成某成就，此刻你需要换一个系统，以寻宝者思维替换职员思维。
  • 人的一生之中，影响最深的可能是几本书，决定前途的可能是一两项技能，改变命运的可能是几件事情，人生贵人也许就那么几个。错过他们的损失不可估量，识别机遇、抓住机会，我们要把主动权攥在手里。
    • 其一，选择了潜在好东西，并为之努力奋斗，并争取做到最好。这需要提高判断力和做事能力，为了提高成功率，需要学习和模仿，行业内顶级大佬就是最好的老师。
    • 其二，必须放量，别指望奇迹。量变是质变的原始积累，并且每一份量变都具有意义。
