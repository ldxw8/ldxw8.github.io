一般观念认为，记笔记是为了纪实，比如记录知识、事件或待办事项，这往往局限于静态理解的范畴。若我们转换视角，以动态眼光重新审视，笔记的价值远超乎想象。以笔记作为枢纽，通过记载构搭建系统，形成一套有章可循、始终闭环的方法体系，以指导我们管理时间安排日程，投资精力专注干事，掌握知识构建体系，复盘思辨总结经验等。

佛畏系统：借以系统思维助我走出人生无序状态

发表于 2024-01-14   |   更新于 2024-04-16 | 分类于 Reading | | 阅读次数:
字数统计: 5,776

凡夫畏果，菩萨畏因，佛畏系统。

  • 先谈两种认识世界的思维模型：

    • 因果思维：借助因果去认识世界，理解事物因果，让人未来有打算、做事情有预案，不至于对什么都感到迷茫。理解因果关系，不沉溺于事后结果，执因索果，透过现象看穿事物本质。然而，真实世界是复杂的，事物发展结果可能由多个归因导致，也有可能因果关系环环相扣，说不清道不明起因、主因亦是常事，何况真实世界还充满了不确定性，即因果知识不能保证我们全然掌控局面，从而科学地做决策。
    • 系统思维：系统性思维，让我们可以宏观地、辩证地、长远地看待每一事物的发展趋势，抓系统规律、辨共性问题、设计系统解决方案，最终做出高质量决策以引导事物往积极方向发展。系统是一个整体、一个组织、一个框架，允许事情在一定范围内变化，怎么变脱离不了一定框架。拥有系统，每发生的一件事只不过是一个个例、一个数字而已。面对一系列相似的问题，都可建立系统去应对解决。

  • 系统思维有助于提升我们认识、理解和解决问题的能力，具体实操又分两方面去执行：

    • 借鉴他人系统：本书讨论了五大类系统，职业发展系统、学习研究系统、日常做事系统、情感人生系统以及社会大系统。借鉴本书建议直接上手实操与验证，比如职场生涯如何规划发展、怎样做好一个领域的研究、怎样摆脱日常事务的枷锁等。
    • 创造个人系统：每个人都需要系统思维，书中方法只是一个引子，将来遇到的每一类问题都可建立一个应对系统。初创系统只是 1.0，未来还要不断打磨这套系统，以致于在面对不确定性时，有方法、有底气做出属于自己的科学决策。

冯友兰：中国哲学简史

发表于 2023-03-18   |   更新于 2023-05-25 | 分类于 Reading | | 阅读次数:
字数统计: 4,143

哲学源于西方的概念名词，各哲学家对于“哲学”的定义亦有不同，中国哲学无不例外。中国哲学，实质上是各思想家学问主张，比如儒家学说、道家学说等。从古至今，中国哲学家特别注重人事之故，极力追求 “内圣外王” 之道，对于宇宙论的研究亦甚简略，若是以西方哲学形式上的系统来定义，那么中国无哲学可言。究竟是对标西方哲学（宇宙论、人生论、认知论），还是自成中国哲学体系，或许这就是冯友兰先生撰写中国哲学史的用意所在，即哲学史要义是找出中国实质的哲学系统。

我们该如何学习中国哲学？冯老先生的《中国哲学史》给出了启示。哲学是对人生终极问题的系统反思，哲学与哲学家出生年代、出生经历、政治背景等有着强相关关系，即哲学具备鲜明个人特点，个人欣赏某一类哲学与他本人的体质、年纪、环境、遭遇等有关系。在思辨过程中，希望你我都能辩证接纳每一种学派思想，无用之用，将来为我所用。
