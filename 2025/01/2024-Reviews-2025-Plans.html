回顾 2024，启航 2025

发表于 2025-01-01   |   更新于 2025-01-19 | 分类于 Journal | | 阅读次数:
字数统计: 10,193

人生已过一世，学艺也有数十载，每逢切换赛道、人生转折点时，心中总会涌起迷茫与不知所措。与其过度思考、内耗焦虑，不如尝试聚焦于一个领域深耕细作，三年为期让知识变现事业走上正轨。

在此过程中，关键在于：有效落实行动，专注于一件事并力争有所建树。聚焦主要矛盾，切莫迷失在知识迷境中不可自拔。

去年年度回顾

  • 汇演播报小助手：探索 AIGC 事业的开篇之作

    • 情况：项目一阶段目标是打造一款拟人化演讲工具，其集自主训练人声模型、生成台词以及合成语音等功能于一身，致力于通过智能化的方式实现幻灯片播报、演讲与视频录播的一体化方案。截至当下，项目已取得了阶段性成果，但距离交付一个令人惊喜、具有价值的最小可用产品，仍需持续迭代优化。
    • 原因：产品形态发展是一个关键问题，它直接关系着产品的消费受众和市场定位。作为一款以幻灯片为媒介实现创作活动的工具，若从第一性原理出发去理解制作幻灯片的本质，那么该工具的价值就在于帮助创作者更高效有质地传达观点、呈现效果以及演讲表达。既然目标是在这些方面致力为创作者赋能，若不能交付一个极小学习成本的产品方案，客户难以触达及认知产品价值，如此一来此前所有努力将可能功亏一篑。

    • 打算：沿着 “两个方向” 努力迈进，即打造人性化交互体系和搭建直觉化创作方案。

      • 其一，人机交互是理解应用内涵的重要途径，着手调研 Python WebUI 框架，以界面交互形式实现产品功能。尤其在用户体验方面，要尽量还原播报幻灯片的交互逻辑，力求做到开箱即用的成效。

        Python WebUI 框架：Streamlit、Brython、Remi、Pywebio

      • 其二，绘声绘影是演绎传达观点的绝妙方式，利用幻灯片演绎动画可低成本地实现此过程。调研动态幻灯片操作与录制的实现方案，搭配自动播报与智能演讲功能，以此创造一个具备竞争力的创作方案。

      • 初拟定 2025 年度二季度有交付成果，于 Github 与小红书同步发布工具产品及宣导作品。

        宣导作品：针对不同用户群体制作一系列宣导内容，这些内容将结合日常创作场景，深入探讨如何以最低成本、高效优质地创作视频。其中，我们要设计巧思引导用户感知产品价值，比如学生汇报课题、老师设计课件、商家介绍产品等，有了该工具改稿件、零剪辑、自配音、添字幕都是一气呵成的事。即通过宣导作品反映产品价值，而非一味自夸产品如何好来赢得用户青睐。

  • 博客 2.0 改造计划：

    • 情况：本计划是全面焕新自身的知识输出结构，让知识分门别类、文章集结成册，给粉丝读者营造一种全新体验。然而，在实施过程中不断思考与反思，意识到仅迭代升级博客并没有触及问题本质。于是回归一个大原则，重新审视当前行动的目的及意义，思考己方能为粉丝提供哪些产品或服务。在此背景下，进而将博客改造计划升级为更为宏大的 “个人品牌化与产品化” 行动。

    • 原因：

      • 其一，不愿做知识的搬运工，缺乏思考与创新，既对自身成长无感，也没产生真正价值。撰写再多的笔记也不过是徒增忙碌之感罢了。
      • 其二，博客改造看似表象调整，但实际是对自身能力以及价值供给的再思考过程。思考赋予何种独特价值，方能撬动用户增长杠杆，实现创作闭环与可持续发展。

    • 打算：参考现有 Things 的知识框架，围绕 “四大形态“ 创作知识产品。其次，一定要大胆将这些产品、作品推向市场并深度推广，毕竟市场是产品调优的最好指导老师。

      在创作过程中，严格遵循 H・K・R・R 原则，即让用户感受快乐，精准传递知识，把控舒适节奏，引发读者共鸣。

  • 个人品牌事业之 小红书起号

    • 情况：小红书作为品牌事业启航的重要渠道之一，战略规划、成长路线、实操打法以及作品规划均已就绪，然而起号进程却稍显乏力。归根结底，还是创业事业没有达到心流状态，创作成果也未能获得积极的回馈，这无疑是保持创作活力的一大障碍。
    • 不足：
      • 至今笔记作品仅有七份，月均不足一份，量级远远不足以支撑账号快速成长。
      • 忽略了周边笔记的生态建设，没能好好利用热门笔记的长尾价值。
      • H・K・R・R 原则掌握得不够火候，要明白优秀内容不局限于作品形态，无论是文字、图文还是视频作品，都具有引发受众共鸣的潜力。
    • 打算：以现有热门笔记为核心，裂变衍生一批周边笔记。抓住一个点，引爆一个点，具体化例子，加深用户对本账号某一类作品的记忆点。待汇演播报小助手取得阶段性小成果之际，也将顺势开启视频创作新篇章，进一步丰富内容形式！

今年事业布局

以创业姿态涣新职场

常以 “管理者” 身份去规划、落实与评估工作，埋头苦干同时勤于思考，从日常工作中琢磨变现、复利及规模化之法，主副业联系与统一，为日后实现 “有钱有闲” 的梦想奠定基础。如此一来，所有思考与行动都应紧密围绕这一核心目标展开，伴随着事物的发展，人生事业与公司事业自然会相互绑定、相互影响、相互成就。

正如标题所言，以创业姿态涣新职场，实质是积累创业所需的心态、认知与能力，为干成一番事业奠定基础。以 “道法术器” 的思想为指引，开启我的创事业征程。

  • 道：主副业并驾齐驱，主业专攻专业、业务与交付能力，副业验证想法、创造无形资产与知识变现复利。

    • 主业：为安身立命，以创业姿态焕新职场，身体力行为我公司创造价值。

      先他人后自己，借公司势成已事业，积极的、有意义的事只管干就完了！

    • 副业：自由之梦想，创富须拥有生产资料，个人思想具象化为有用户受众、有消费潜质的产品。

  • 法：构筑一件伟大的事物，需要广泛而深入的多维度认知，以及丰富的领域知识积淀。这是一个认知与实践反复交融、螺旋上升的过程，联动 “五大能力” 参与其中、共同建设。

    五大能力包括产品、运营、设计、研发、数分。产品是检验能力的标准，运营是价值变现的手段，设计、研发、数分三大能力融合赋能，最终五位一体共同塑造你的独特竞争生态位。着重思考，当前阶段能力怎么发力于职场、往后如何发挥能力打造消费级产品。

  • 术：道篇章坚定了未来 3-5 年的奋斗目标，法篇章界定了职业发展方法论，术篇章则给予思想行动指南。

    • 提升专业能力，自研无形资产，发布公众媒体，用户买单变现。
    • 抛开以往认知，撰写个人传记，提炼工作能力，突出个人优势。
    • 定赛道选岗位，按难度、意愿、匹配三个维度，再次扬帆起航。

  • 器：有预见性地行动起来，直面困难解决问题，经历才能预见下一步动作。

    • 自研产品：产品是自我认知水平及专业能力的具象反映，可设定其为本计划的北极星指标。
      • 移动应用，为超级个体打造一款基于笔记方法论的时间、精力、知识管理系统。
      • 汇演助手：一款结合文字、声音、图像领域的 AIGC 产品，分阶段建设三个产品形态，先是绘声，再是绘影，最终建成内容创作者讲好故事的辅助神器 — 自生文、文生图、文发声。

    • 数字化

      • 零售转型：人（人效）+ 货（利润） + 场（坪效）；客户（营收）+ 产品（中收）+ 渠道（成本）
      • 对公转型：用户画像 + 估值体系 + 服务设计

    • 小红书：以教学相长之道，分享所学所悟、思维想法或经验，形成图文笔记并公布于众。

    • 撰写自传：以撰写简历为范例，对过往经历、所做所为的阶段总结与再反思。简历代表着个人门面，借重点项目彰显优势，并详述项目细节以印证实力。其中，剖析项目要融入己见与思考，佐以辩证分析过程，绝不止于纸面几句话草草了事。

个人品牌化与产品化

以 “我” 作为品牌化的枢纽，瞄准受众，抓住诉求，融合所有能力共同铸就某一件产品或服务，最终为客户提供独特价值。

个人品牌化与产品化顶层框架 2.0

  • 总体原则：审视该行动的目的及意义，思考己方能为粉丝提供哪些具象化的产品或服务？

    比如：做一件产品，你的目标受众是谁？通过“三个小”战略，先聚焦于精细化客群，再提炼共性痛点诉求，最后发力开发最小可迭代产品（MVP）。

  • 定义品牌：实质上是自身优势及价值供给的具象化过程。以个人品牌为例，愿景是融合工程、美学、数据等多个学科思维及能力，打造与之呼应的知识产品与效率工具，助力你我成为新时代的优秀创作者。

  • 建立指标：建立个人品牌不是一句空话，也不是闭门造车、自嗨创业，客户认可这个品牌先决条件是认可旗下的产品或服务。品牌定位必须统一目标，制定北极星指标，以引领产品或服务的设计、开发与拓客过程，不偏不倚、聚焦主题。做不到这些也只是忙里忙去，感动了自己。

    借鉴影视飓风的 HKRR 方法。优质内容同时要具备四种元素（HKRR），快乐（Happiness）、知识（Knowledge）、共鸣（Resonance）、节奏（Rhythm）。有了指标量化，就得有手段去发现问题、改善情况，比如服务设计领域的顾客旅程图，将体验内容的关键节点給标识出来，明显哪个节点没有良好反馈、没有情绪爆点等就得改进。

  • 产品布局：限定为 App、项目经验、技术思考、思维课堂四种形态。它们承载着不同战略思考，在时间资源与精力资源有限情况下，不开疆扩土、不盲目追新，专心研发产品并聚焦品牌建设。

    • 承载方法理念的应用：时间、精力、知识管理应用；AIGC 相关组件、解决方案
    • 变现实践经验的产品：图文课堂、视频课堂，主打项目经验、技术思考及思维模型相关内容分享

  • 能力变现：自身所拥有的从业经验、项目思考以及专业知识，既是己方的能力边界也是独特的知识财富。与 以创业姿态涣新职场 项目给联动起来谈。

    • 该项目核心任务是复盘过往项目经历，思考并总结己方发挥的价值（有无可复用的 SOP 产出）。
    • 涉及什么专业知识，以教学相长方式理解并讲述技术栈。
    • 最终，项目经验与技术思考形成知识产品（博客重要组成部分）。至于什么形式呈现，只要不过分稀释资源的、有助于完成最终目标的，都可以大胆去做。

  • 渠道基建：无论是知识产品，还是应用产品，新兴产品都需借助三方平台去营销传播。初期主要矛盾是产品能否带给客户价值，至于客户来不来博客消费内容不是本阶段关心的重点。但自家渠道又不能没有，大本营整合了所有产品资源，是一个方便客户集中消费内容的渠道之一。至于怎样建设渠道，请按照不同阶段矛盾、客户诉求以及服务体验来深入探讨。

    • 必要前提，内容必须足够优秀，做营销一定要相信产品是优秀的。
    • 具体打法，优先考虑受众是谁，再考虑产品的传播形态，最后选择合适的投放平台。
    • 通俗理解，类比吃饭先确定吃什么菜式再找饭店，自己吃得满意、饭店获得好评，选择对的渠道会叠加传播效果。

  • 客户受众：留在最后谈不是不重要，反而是整个项目的根本。以结果为导向，客户消费内容，规模化价值与变现。客户不买单，整件事只会感动自己。因此，必须事先用户调研，再思考怎么做好产品。

    客户是内容创作者，还是内容分享者？需再认真思考。

认知创业与方向探索

  • 新零售体系：瞄准三大主体，设定三大目标，理解领域概念，以尝试建立一套新零售经营体系。

    • 银行业：客户、产品、 渠道；营收、中收、成本
    • 传统超市：人 、货、场；人效、利润、坪效
    • 知识变现：未来超级个体会越来越多，普通个体应如何积累数据及经验，并结合新零售思维以开启知识变现生意，这也是值得探究的问题。

  • 运营体系：用户运营、产品运营、生意运营

万卷书万里路

回首三年前立下的目标，彼时怀揣满腔的好奇心，广泛涉猎各类书目，力求拓展自身的认知边界。如今，主业工作愈发忙碌，迫使我不得不对书目进行调整，将有限的精力投资于现阶段最具实用性的知识领域。比如，想攻坚新零售领域，书目将集中在经商管理、数据科学等。

另一层面，即使再忙专业技能不能丢，要始终有深度地耕耘与迈进。在技术架构、机器学习、深度学习不断深造，同时结合自身的工作岗位及能力成长需要，强化并巩固数据分析、数据工程方面的核心竞争优势。2023 年入局，2024 年入微，2025 年争取在金融及零售领域略有小成。

最后，回顾一遍读书清单，分享一些从中获得的有趣启示，愿能与大家探讨心得、共同进步。

  • 2024.12 | 产品 |《真需求》

    • 共识：做一款产品，价值不成立最好选择是早死早超生。价值成立但市场不动，那一定是感知、想象、场景、利益相关人四个环节中，至少一个出现了卡壳。
    • 共识：产品匹配场景，才能产品匹配市场。受益者驱动做成一件事，受益者越多越易做大一件事。如果事情进展不顺利，多数是利益阵营问题。
    • 共识：一款新产品，如果没确定哪个场景去发力，可以先进行场景穷举，再进行场景识别。
    • 共识：《鬼谷子· 忤合》的认知框架教识我们如何与人建立共识，动以真情，强化信任，持续忤合，赢得共识。同理，工作中说服领导、新市场引导用户等，需要给足时间与空间让 “共识” 种子萌发。
    • 共识：用户选择一款产品，获得了功能价值。而我们做的不仅如此，更高层次做法是让客户知道选择该产品是多么明智的选择，提供一波情绪价值。
    • 模式：保留应对不可抗风险的能力，为事业留有足够翻身的安全边界。比如，保命现金储备、冗余运作机制。
    • 模式：强管理与强激发，奖励与惩罚并行齐驱。利用人的损失厌恶心态，失去比得到这些更难受，“反其道而动” 考虑绩效分配机制。
    • 模式：企业文化决定了企业管理目标及管理过程细节，创新只允许成功不允许失败，这样的创新也徒有虚名。

  • 2024.09 | 理财 |《金钱心理学》

    • 低物欲高储蓄，这是变富有的先决条件（高风险高回报的投资策略并不适宜普通大众，不在此列）。剩下就是相信复利的力量，无论是生产力提升、知识储备扩充、财富积累或社会关系延展，它们都会随着时间叠加在未来兵迸发超乎想象的能量。当今时代，消费主义甚嚣尘上，享乐之风肆意弥漫，各类 “小额失血” 式的消费陷阱层出不穷，想堕落没人拦你，但你想出人头地，拦你的人海了去，变富路上请勿忘记初心。

  • 2024.08 | 效率 |《番茄工作法图解》

    • 番茄工作法，核心价值是帮助人们提升专注力，当人处于专注状态时，更容易获得心流体验，进而提高做事效能与效率。回顾该方法本身，关于如何设计原子性番茄钟，本质就是考究我们如何拆解小目标的能力。以最小可持续执行单元内专注做事情，在过程中不断训练与调整，直至你获得一种保持高度聚焦的最佳体感，此刻番茄钟便可指导我们做好任何事情。

  • 2024.06 | 效率 | 《笔记的方法》

    综合《子弹笔记》《时间管理》《番茄工作法》《OKR 工作法》等，总结一套成就超级个体的笔记方法论。

    • 放不下面子，得彻底改观自己的认知，才能从事营销工作！
    • 把握自愿享受福利的顾客，一定让客户尝到 “足够” 甜头，寄望于此带来链式传播、复利效应。
    • 通过被动接受福利的顾客，特别是第一轮引流过来的顾客，讲究快刀斩乱麻，先入为主、入袋为安，再缓缓地供给好服务。
    • 主动营销意愿低的客户，要有耐心、为顾客着想，见人说人话，投其所好不急着营销。找准时机切入福利营销。
    • 没意愿的客户就不浪费时间、人力成本，除非真的一点资源都没有了。

  • 2024.02 | 思想 | 关于情绪价值理解

    • 自开启小红书创副致富之路以来，有一个不断强调与反思的问题：无论是竞品调研也好，还是自身供给也好，一款笔记或一个视频要传递给观众一份价值，观众接纳了这份价值，才会认可你的内容，或更进一步认可你。价值有情绪价值、工具价值之分，且它们有优先顺序。
    • 进一步谈谈什么是情绪价值呢？以目前认知去理解，姑且称它为一种主客观感受的统一体吧。喜怒哀乐悲恐惊，一方显现某一种情绪，另一方能否产生共情或共鸣，双方要有意愿参与其中才有效果，价值供给方营造机会、给出提示，引导价值接收方理解接纳，双方主动交流接触，直至双方节奏逼近，便是双方情绪感受的主客观统一顶峰时刻。先共情再共鸣，此刻才显现情绪价值。
    • 两个人互相提供情绪价值，实质就是认知、经历、价值观相仿的两个人互相寻求认同感的过程。
